Frame shaka::media::VideoRenderer::DrawFrame ( int *  dropped_frame_count,
bool *  is_new_frame,
double *  delay 
)
overridevirtual

For a video renderer, this will draw a frame to a texture and return it. For other renderers, this returns an invalid frame.

Parameters
dropped_frame_count[OUT] Will contain the number of frames that were dropped since the last call to DrawFrame.
is_new_frame[OUT] Will contain whether this frame is a different frame than the previous call to DrawFrame.
delay[OUT] Optional, will contain the delay, in seconds, until the next frame should be drawn.
