Detailed Description

Defines a buffer of media frames. This is stored as a series of contiguous blocks of buffered ranges.

This type is fully thread safe. Any thread that gets frames MUST only use the frame through the Guard instance to ensure that another thread doesn't delete the frame while it is in use.

§ Remove()

void shaka::media::FrameBuffer::Remove ( double  start,
double  end 
)

Removes the frames that start in the given range. This will remove frames past end until the next keyframe to mirror MSE requirements.

Also to mirror MSE, this always uses PTS to determine which frames to remove. This will mean that some frames before |start| may be removed because they depend on removed frames.

If other threads are using frames from this buffer, this will block until they are no longer being used.
